# Clustering K-Means Penyelamatan Darurat Kota Semarang

Aplikasi web untuk clustering tingkat intensitas penyelamatan dan evakuasi 
darurat di Kota Semarang menggunakan algoritma K-Means.

**Program Studi:** Sistem Informasi, Universitas Negeri Semarang  
**Periode Data:** 2023–2025  
**Sumber Data:** BPS Kota Semarang — Kota Semarang Dalam Angka 2026

## Cara Menjalankan

pip install -r requirements.txt
python app.py

Buka: http://localhost:5000
